# Todo List

A simple Todo List application built using HTML, CSS, and JavaScript. This project is made for DOM manipulation practice and mastery.


# Features

* Add new tasks
* Checkbox to select a tesk
* Prevent empty input
* Delete taks
* Edit tasks
* Clean UI


# Tech Stack

* HTML
* Tailwind Css
* JavaScript (DOM Manipulation)

# Key Concepts Used

* `document.querySelector()`
* `addEventListener()`
* `insertAdjacentHTML()` / `append()`
*  Input validation using `.trim()`


# Learning Outcome

This project helped in understanding:

* How JavaScript interacts with HTML using the DOM
* Event handling in real applications
* Dynamic content rendering
